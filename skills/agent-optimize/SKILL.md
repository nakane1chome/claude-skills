Optimize the target document for agent consumption.

**Stop after completing and have changes reviewed with user.**

1. **Replace verbose prose with structured formats**
   - Convert narrative paragraphs to tables, diagrams, or bullet lists where meaning is preserved
   - Use structured formats agents parse well (clear headers, consistent patterns)

2. **Compress token usage**
   - Remove redundant phrasing without losing information
   - Prefer concise, direct language over verbose explanation

3. **Make relationships explicit**
   - Add cross-references between related sections and documents
   - Use hierarchy indicators to show parent/child relationships
   - Prefer visual structure over narrative explanation
